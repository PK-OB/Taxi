현재 flutter를 경험해보고 싶어 간단한 어플을 구현중입니다.
(chatgpt와 유튜브 강좌 사용중)

환경 > php (vsc), flutter dart (android studio) , mysql, xammp (DB)

api_app폴더는 xampp/htdocs/api_app경로의 폴더

api_app/connection.php 는 mysql 데이터베이스와 연결 담당

api_app/user/signup.php 는 데이터베이스의 user테이블에 정보를 저장하기 위한 코드

api_app/user/validate_email.php 은 user 테이블 중 중복되는 user_email이 있는지 식별


lib 폴더는 flutter 프로젝트 내의 폴더

lib/api/api.dart 파일은 api_app폴더 내의 파일들과 android studio의 프로젝트를 연결하기 위한 파일

lib/authentication/signup.dart 파일은 사용자로부터 회원가입 정보를 입력받아 전달해주는 파일
> signup.dart 파일 내 saveInfo 를 통해 user 클래스에 데이터를 전달

lib/model/user.dart 파일은 사용자 정보를 담을 변수가 저장되어있음


이 코드중 공통적으로 쓰이는 코드는
connection.php 
api.dart
이며

signup에 쓰이는 코드는 
signup.php
validate_email.php 
signup.dart
user.dart
코드입니다.

이 방식으로 signup 기능은 문제없이 작동하고 있으며, 똑같은 방식으로 alarm 기능을 구현하려
공통적으로 쓰이는 코드에

api_app/user/alarm.php (signup기능에서 signup.php에 해당)
api_app/user/valiate_alarm.php (signup기능에서 validate_eamil.php에 해당)
lib/writeAlarm.dart (signup기능에서 signup.dart에 해당)
lib/model/alarm.dart (signup기능에서 user.dart에 해당)
파일을 추가하여 구현하려 합니다.

현재 writeAlarm.dart의 생성 버튼을 눌러도 데이터베이스에 데이터가 저장이 되지 않는 오류를 해결하지 못하고 있습니다.

몇가지 테스트를 거쳐 writeAlarm 코드 내 saveinfo가 제대로 작동하지 않는것을 확인했습니다.(이걸 원인으로 추측중)
어떠한 이유로 signup기능은 제대로 작동하는데 alarm기능은 작동을 하지 않는지 알고싶습니다
